📈 AI Stock & F&O Analyzer
An advanced AI-powered Stock & F&O Analysis tool built with Streamlit, yFinance, Pandas, and Plotly.
This project combines technical indicators, smart money concepts, candlestick patterns, and volume analysis into a single interactive dashboard — designed for learning, research, and strategy development.

⚠️ Disclaimer: This tool is for educational purposes only. It is not financial advice.

🚀 Features

🔍 Multi-Mode Analysis
Stock Analysis: Enter any NSE ticker (e.g., INFY.NS, TCS.NS, RELIANCE.NS).

F&O / Index Analysis: Preloaded with popular indices and stocks (NIFTY, BANKNIFTY, SENSEX, Reliance, Infosys, etc.).

📊 Technical Indicators

Moving Averages (MA20, MA50)

Relative Strength Index (RSI)

MACD (Momentum Confirmation)

VWAP (Institutional Levels)

Bollinger Bands

ATR (Volatility-based Stop Loss)

🧠 Smart Money Concepts

Swing Highs & Lows

Break of Structure (BOS)

Change of Character (CHoCH)

Order Blocks

Fair Value Gaps (FVG)

🕯️ Candlestick Pattern Recognition
Bullish: Hammer, Bullish Engulfing, Morning Doji

Bearish: Shooting Star, Bearish Engulfing, Evening Doji

📈 Signal Generation Engine
8-layer confirmation system:

RSI + MA (base layer)

MACD confirmation

VWAP positioning

Support/Resistance zones

BOS/CHoCH structure

Order Blocks

Candlestick Patterns

Volume Spike filter

Signal Strength: STRONG / MODERATE / WEAK

🎯 Trade Setup
Dynamic Stop Loss using ATR + Swing Points

Auto-calculated Target Price (TP) and Stop Loss (SL)

Adapts to volatility (tight SL for stable stocks, wider SL for volatile ones)

🖥️ Interactive Dashboard
Built with Streamlit for real-time visualization

Auto-refresh option (5s) for live monitoring

Clean UI with charts powered by Plotly

🌟 Benefits
All-in-one toolkit: Combines classical indicators with modern Smart Money Concepts.

Educational focus: Perfect for students, traders, and researchers learning technical analysis.

Dynamic risk management: ATR-based stop loss adapts to volatility.

Visual clarity: Interactive charts make complex signals easy to understand.

Customizable: Extend with new indicators, patterns, or strategies.
